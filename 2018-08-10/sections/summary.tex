\section{Summary}

\frame{
   \frametitle{Summary I}
   \begin{itemize}
      \item \textcolor{red}{$(V-A)_L$} nature of the \textcolor{red}{hadronic weak charged current} is still an open question.
      \vspace{0.4cm}
      \item Can be probed via exclusive \textcolor{red}{$\Bbar \to \xcu \ellm \barnuell$ angular} analyses
      \vspace{0.4cm}
      \item Full \textcolor{red}{hadronic tagging} is a \textcolor{red}{powerful} tool at (super)-\textcolor{red}{$B$-factories} to probe this by giving direct access to $\barnuell$ 4-momentum.
      \vspace{0.5cm}
      \item \textcolor{red}{First} ever 2-D analysis of \textcolor{red}{$\Bbar \to D \ellm \barnuell$} w/ \textcolor{red}{$\{\qsq,\ctl\}$ correlations} in $S/B$ separation and acceptance correction accounted for.
      \vspace{0.5cm}
      \item Extending this to the first ever \textcolor{red}{tagged} full \textcolor{red}{4-D} analysis of \textcolor{red}{$\Bbar \to V \ellm \barnuell$}, $V\in \{D^\ast, \rhoz\}$
   \end{itemize}
}


\frame{
   \frametitle{Summary II}
   \begin{itemize}
      \item \textcolor{red}{$\Bbar \to D^\ast (\to D\pi) \ellm \barnuell$}: uniquely \textcolor{red}{clean} environment at $\babar$.
      \vspace{0.4cm}
      \item  $\textcolor{red}{\delta(\epsR)} \sim \mathcal{O}(\textcolor{red}{15\%})$ sensitivity to \textcolor{red}{RH} admixture might be feasible
      \vspace{0.4cm}
      \item Simultaneously, work on the more complicated \textcolor{red}{$\Bbar \to \rhoz \ellm \barnuell$} ongoing.
      \vspace{0.4cm}
      \item Analysis is in mature state and under review. Final results expected soon.
      \vspace{0.4cm}
      \item Methods developed and tested here can be directly applied to other \textcolor{red}{$B \to VV$} or \textcolor{red}{$B\to \{P,V\}\ell \ell$} analyses using the high statistics \textcolor{red}{LHCb} dataset.
   \end{itemize}
}
